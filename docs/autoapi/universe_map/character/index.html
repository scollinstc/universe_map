name_type¶

Specifies what type of name is associated with this name for a character. Includes types like ‘Familiar’, ‘Alias’, ‘Formal’, ‘Legal’, etc.

Parameters:

  • name_types (list) – List of name types, with indices corresponding to the names given in the names parameter.
